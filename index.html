def get_order_items(order_id):
    order = Order.objects.get(id=order_id)
    items = []
    for item_id in order.item_ids:
        items.append(Item.objects.get(id=item_id))
items = Item.objects.filter(id__in=order.item_ids)
    return items
N+1 query — this loop issues one database call per item, roughly 40 per request under current order sizes. The suggested change fetches every item in a single query. Ready to review in your next pull request.
